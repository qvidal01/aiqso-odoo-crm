# aiqso-odoo-crm

Utility scripts for importing and syncing lead data into Odoo 17 CRM via XML-RPC

## Installation

```bash
git clone https://github.com/qvidal01/aiqso-odoo-crm.git
cd aiqso-odoo-crm
```
