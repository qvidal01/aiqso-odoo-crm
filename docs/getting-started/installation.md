# Installation

## Requirements

- Python 3.9+
- Dependencies listed in `requirements.txt`

## Install from PyPI

```bash
pip install aiqso-odoo-crm
```

## Install from Source

```bash
git clone https://github.com/AIQSO/aiqso-odoo-crm.git
cd aiqso-odoo-crm
pip install -e .
```

## Optional Dependencies

```bash
pip install aiqso-odoo-crm[all]
```
